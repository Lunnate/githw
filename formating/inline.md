# Моноширинный 

* Моноширинный шрифт — все символы занимают одинаковую ширину, как в терминале.

## Назначение:

Для выделения фрагментов кода, команд, конфигураций, переменных в тексте

## Синтаксис: 

```markdown
`code`
```

### Пример:

Чтобы вывести текст, используйте команду `print()`
