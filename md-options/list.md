# Списки 

Markdown позволяет создавать два типа списков: маркированные и нумерованные.

## Маркированные списки 

Маркированные списки — это списки, элементы которых начинаются с маркера, обычно `-`, `*` или `+`.

### Синтаксис:

```markdown
- Элемент 1
- Элемент 2
  - Подэлемент
* Элемент 3
+ Элемент 4
```

### Пример:

- Молоко
- Хлеб
  - Цельнозерновой
* Яйца
+ Сыр


##  Нумерованные списки

### Синтаксис

```markdown
1. Элемент 1
2. Элемент 2
   1. Подэлемент
   2. Подэлемент 2
3. Элемент 3
```

### Пример:

1. Молоко
2. Хлеб
   1. Цельнозерновой
   2. Белый
3. Яйца
